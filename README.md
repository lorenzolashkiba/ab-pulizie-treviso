# AB Pulizie Treviso - Sito Web

Sito web professionale per servizi di pulizie a Treviso e provincia.

## Caratteristiche

- Design moderno e responsive
- Ottimizzato per SEO locale (Treviso)
- Form di contatto per preventivi rapidi
- Mobile-friendly
- Velocità di caricamento ottimizzata
- Logo personalizzato AB

## Struttura del Progetto

```
Pulizie/
├── index.html          # Pagina principale
├── css/
│   └── style.css       # Stili CSS
├── js/
│   └── script.js       # JavaScript per interattività
├── img/                # Cartella per immagini
├── robots.txt          # File per crawler
├── sitemap.xml         # Sitemap per SEO
└── .htaccess           # Configurazione server
```

## SEO Ottimizzazioni Implementate

1. **Meta Tags Ottimizzati**
   - Title ottimizzato con keywords locali
   - Meta description accattivante
   - Open Graph tags per social media
   - Keywords mirate per Treviso

2. **Contenuto Ottimizzato**
   - Heading structure corretta (H1, H2, H3)
   - Keywords naturalmente integrate
   - Focus su "pulizie Treviso"
   - Menzione di zone specifiche (Villorba, Silea, etc.)

3. **Performance**
   - CSS e JS ottimizzati
   - Compressione Gzip abilitata
   - Browser caching configurato
   - Lazy loading implementato

4. **Local SEO**
   - Menzione chiara della zona operativa
   - Liste di comuni serviti
   - Informazioni di contatto visibili
   - Schema markup ready

## Come Usare

1. **Personalizzazione Dati**
   - Sostituire numero di telefono: `+39 333 123 4567`
   - Sostituire email: `info@abpulizietreviso.it`
   - Aggiornare P.IVA nel footer
   - Aggiungere indirizzo reale se necessario

2. **Aggiungere Immagini**
   - Inserire immagini nella cartella `img/`
   - Foto del team, lavori eseguiti, etc.
   - Ottimizzare le immagini per il web (compresse)

3. **Form di Contatto**
   - Attualmente il form mostra solo un messaggio di successo
   - Per funzionalità reale, integrare con:
     - Servizio email (EmailJS, FormSubmit, etc.)
     - Backend PHP
     - Servizi come Netlify Forms o Formspree

4. **Hosting**
   - Caricare i file su hosting web
   - Configurare dominio personalizzato
   - Installare certificato SSL (HTTPS)
   - Attivare redirect in .htaccess

## Prossimi Passi per SEO

1. **Google My Business**
   - Creare profilo Google My Business
   - Aggiungere indirizzo, orari, foto
   - Raccogliere recensioni

2. **Local Citations**
   - Registrarsi su PagineGialle.it
   - Profilo su TuttoCittà
   - Directory locali di Treviso

3. **Contenuto**
   - Aggiungere blog con articoli su pulizie
   - Creare pagine specifiche per ogni servizio
   - Gallery di lavori eseguiti

4. **Analytics**
   - Installare Google Analytics
   - Configurare Google Search Console
   - Monitorare posizionamento keywords
